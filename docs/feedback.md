# Feedback

Feedback, bug reports and feature suggestions are always welcome.

Either report an [issue](https://github.com/mickeytheq/hades/issues) in Github or post in [this discord channel](https://discord.com/channels/225349059689447425/1450141169639096402).

For bug reports please provide a full list of steps to reproduce and a full copy of the contents of any relevant log from the progress dialog.
